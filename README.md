\# Lab 4 - Git Practice

Це лабораторна робота з Git.



\## Мета

Навчитися користуватися Git: клонування, створення гілок, коміти, push, pull request.



\## Виконані кроки

\- Створено репозиторій на GitHub

\- Клоновано репозиторій локально

\- Додано README.md

\- Створено гілку feature-readme

\- Виконано коміти і push



\## Використані команди Git

\- git clone

\- git branch

\- git checkout

\- git add

\- git commit

\- git push

\## Нові зміни

\- Додано додаткову інформацію про Git

\- Пояснення команд Git для лабораторної
